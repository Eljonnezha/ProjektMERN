# EatNow 

EatNow është një aplikacion web për porositjen e ushqimit online, i ndërtuar me **MERN stack** (MongoDB, Express.js, React, Node.js). Projekti është krijuar si pjesë e portofolit dhe prezantimit në kurs.

---

## Qëllimi
- Ofrimi i një platforme moderne dhe të thjeshtë për porositjen e ushqimit.
- Demonstrimi i aftësive teknike në zhvillimin e aplikacioneve web.
- Balancimi i realizmit me thjeshtësinë për prezantim akademik dhe praktik.

---

## Teknologjitë
- **Frontend:** React + Bootstrap (UI responsive dhe moderne).
- **Backend:** Node.js + Express.js (API dhe menaxhim porosish).
- **Database:** MongoDB (ruajtja e përdoruesve, porosive, mesazheve).
- **Siguria:** `dotenv` për menaxhimin e environment variables.

---

## Funksionalitetet
- **Regjistrim/Login**
  - Validim i email-it dhe fjalëkalimit.
  - Mesazhe suksesi/gabimi.
  - Conditional rendering për butona si *View Shport*.

- **Porositë**
  - Klienti plotëson artikullin, sasinë dhe te dhenat.
  - Statusi i porosisë: *Processing* (opsional: *Delivered*).

- **Admin Dashboard**
  - Menaxhimi i porosive dhe mesazheve.
  - Opsione për fshirje/update.
  - Sidebar dhe refresh logjik për mesazhet.

- **UI/UX**
  - Navbar me opsion për AdminDashboard.
  - Paletë ngjyrash moderne për butona (Shport Checkout, Submit, Close).
  - Seksione prezantuese: *Why Choose Us*, *Testimonials*.
